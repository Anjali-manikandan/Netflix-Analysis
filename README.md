# Netflix Data Analysis Project

This project provides an **in-depth analysis of Netflix data**, including subscription trends, top genres, revenue patterns, and content insights. The project includes a **Python notebook** for data cleaning, exploration, and visualization, as well as a **Power BI dashboard** for interactive insights.

## 📂 Project Files
- `Netflix_analysis.ipynb` → Python notebook with full analysis and visualizations  
- `Netflix_dashboard.twbx` → Interactive Power BI dashboard
- 
## 📊 Key Insights
- Most popular genres and top-performing shows  
- Subscription trends over time  
- Revenue patterns and geographic insights  
- Visual representation through an interactive dashboard  

## 🛠 Tools & Technologies
Python | Pandas | Matplotlib | Seaborn | Tableau
